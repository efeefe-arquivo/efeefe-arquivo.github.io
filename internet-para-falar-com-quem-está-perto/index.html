A internet para falar com quem está perto

 
Este post foi agregado pelo meu lifelog. É possível que eu não seja o autor.

A lógica do negócio é: o custo do serviço de banda larga está caindo. Se eu posso ter bastante banda pra mim a baixo custo, porque não compartilhar um pouco dessa banda com as pessoas da minha rua?

Agora, e se ao fazer isso, eu incluir na página inicial de acesso um fórum aberto, para as pessoas da minha rua conversarem umas com as outras?

É isso que este projeto, NeighborNode, desenvolvido no programa ITP da NYU (meu objeto de desejo), oferece. Ele te ensina qual o equipamento de wi-fi voce deve comprar e te dá o programa para gerar o mural quando as pessoas acessarem a Web.

O projeto gerou certo buzz, resultando inclusive em reportagem no NYtimes. Isso foi em 2004. Havia empolgação no ar, os envolvidos falavam de outros degraus para desenvolver o projeto:

“We are presently beginning Phase Two of the project, which will include creating a CD-ROM that allows you to configure any old PC as a Neighbornode instantly, as well as allowing individual nodes to grow dynamically with the use of mesh networking. Phase three, scheduled to begin in 2005, will involve trying out new social uses for this infrastructure, including art projects, political activism, gaming, file sharing, and so forth.”

Mas pelo visto o projeto não andou desde então, as últimas mensagens são de 2005. Mas, apesar de parada, a página está no ar e parece que vai angariando simpatizantes que, seguindo as instruções, introduzem a idéia em suas ruas.

Como existe custo envolvido - voce precisa comprar o equipamento para transmissão wireless -, eles informam que esse aparelho pode ser encontrado a baixo custo no Ebay.

Talvez o motivo disso ter falhado tenha a falta de massa crítica para compartilhar custos. E se isso fosse parte de um pacote de acesso à Web? Se ao invés de vender para indivíduos, o pacote fosse para grupos de vizinhos, e incluísse uma parte da banda para ser usada coletivamente. Eventualmente os free-riders poderiam se interessar em fazer o upgrade. Ou haveria algum tipo de contra-partida nesse mural como o patrocínio do comércio local.

Me ocorreu outra coisa, ainda: isso pode ser feito sem o wi-fi. É só criar o forum na web e mostrá-lo encontrar uma forma para que os vizinhos fiquem sabendo que ele existe. Mas não seria a mesma coisa. Primeiro porque o acesso à rede é um atrativo para que as pessoas entrem no espaço. E também o fato do perímetro de ação ser limitado pelo alcance da transmissão também serve para limitar a entrada especificamente de quem estiver perto, o que nao seria possível de ser controlado se o mural estivesse aberto na web.
